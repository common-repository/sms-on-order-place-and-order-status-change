This is an awesome plugin for woocommece shop. This plugin gives your site the ability to send sms on every order placement. the pro version is cheap enough but allow you to send sms in every order status change. Contact the author Naim-Ul-Hasan for the pro version.

== Description ==
Sending SMS on order placement and in every order status change increases your sales exponentially. 
This helps to gain customer trust easily. once you gain their trust, you are on business. 
This plugin uses Techno71 api only for Bangladeshi users. so go for <a href="https://www.facebook.com/techno71.bd">techno71</a> to get an account. purchase a plan and grab the api key, username and sender Id that's all the plugin needs. 


== Installation ==
Follow the steps below to install the plugin.

    1. Upload the \"woo-sms-for-order-status\" directory to the /wp-content/plugins/ directory.
    2. Activate the plugin through the ‘plugins’ menu in WordPress.
    3. Go to \"woocommerce/Mohaazon Api integration\" to fill up your username and password.
     That\'s it.
